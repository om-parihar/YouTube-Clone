# 🎬 YouTube Clone (HTML & CSS)

A YouTube-inspired web interface built using **HTML and CSS**, focused on replicating the layout and design of the YouTube homepage.

---

## 🚀 Features

- 📺 Video grid layout (like YouTube homepage)
- 📌 Fixed sidebar navigation
- 🔍 Header with search bar and icons
- 👤 Channel info section (thumbnail, title, views)
- 🎯 Clean and structured UI
- 📱 Basic responsive design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox & Grid)
